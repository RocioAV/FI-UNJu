## 📖 Sobre el Proyecto

**FiUnju** es una aplicación web desarrollada para la administración eficiente de instituciones educativas. El sistema centraliza la gestión académica, permitiendo llevar un control detallado de los registros y facilitando el proceso de inscripción a materias para los estudiantes.

Este sistema fue desarrollado como **Proyecto Integrador (2024)** para la Facultad de Ingeniería de la Universidad Nacional de Jujuy (UNJu).

## ✨ Características Principales

- **Gestión Administrativa (CRUD):** Sistema completo para la creación, lectura, actualización y eliminación de registros institucionales (alumnos, materias, docentes, etc.).
- **Módulo de Inscripciones:** Interfaz dedicada para que los alumnos puedan autogestionar su inscripción a las materias de su plan de estudios.
- **Búsqueda Optimizada:** Implementación de filtros de búsqueda avanzados para localizar información de usuarios y materias de manera rápida y precisa.

## 🛠️ Tecnologías Utilizadas

- **Backend:** Java, Spring Boot.
- **Base de Datos:** MySQL
- **Frontend / Vistas:** Thymeleaf.
- **Arquitectura:** Patrón MVC
